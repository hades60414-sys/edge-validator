投放資料夾(watch/)— Edge Validator 本地照妖台
================================================

把任何回測/策略的報酬或淨值 CSV 丟進這個資料夾,重新整理 App 後就會自動出現在
「📁 投放資料夾 CSV」清單裡,選它就能送進同一把統計裁判照妖。零上傳、全本地。

支援格式(與線上公開版偵測邏輯同源):
  1. 單欄       每列一個數字(逐期報酬 或 淨值,自動偵測)。
  2. 日期+數值  兩欄:date,value(value=報酬或淨值,自動偵測;含民國年會轉西元)。
  3. 多策略矩陣 date,策略A,策略B,…:自動挑「樣本內夏普最高」的一欄送審,
                並把 n_trials 設為欄數,讓 DSR 據此通縮(懲罰「挑最好」的過擬合偏誤)。

數字清洗:千分位逗號、百分比(2.5% → 0.025)、前導 +、全形數字都吃得下。
壞掉的 CSV 仍會列出並標明錯誤原因,不會默默消失。

覆寫路徑:設環境變數 EDGE_WATCH_DIR 指到別的資料夾即可。

範例(單欄逐期報酬):
    return
    0.012
    -0.003
    0.008
