# Edge Validator — 策略照妖鏡

**把你的回測丟進來,它會誠實告訴你:這個 edge 是真的,還是你把雜訊擬合成了曲線。**

大部分回測平台幫你「找到看起來會賺的策略」。這一個相反——它假設你的策略是假的,然後用一整套學界標準的統計檢定try to 反駁你。過得了關的才可能是真 edge;過不了的,恭喜你在賠真錢之前就知道了。

## 為什麼做這個

這是一套量化策略農場（4,800+ 次試驗、0 個通過誠實樣本外驗證）沉澱下來的**裁判制度**。農場的結論是殘酷的:免費資料 + 散戶資本下，可部署的 alpha 幾乎不存在。但那套「不放水的裁判」本身有價值——它能在你把錢放進一個過擬合的回測之前攔下你。

## 它做什麼（七道誠實閘）

上傳你的策略**逐期報酬**（或淨值曲線），它跑：

1. **Deflated Sharpe / DSR** — 扣掉「你試了幾種參數才挑到這條曲線」的運氣成分
2. **PBO / CSCV** — 樣本內最優參數在樣本外還站得住嗎（過擬合機率）
3. **Hansen SPA** — 數據挖掘後，你的策略真的贏過基準嗎
4. **Romano-Wolf StepM** — 多重比較家族誤差修正後的倖存篩選
5. **成本壓力測試** — 手續費/滑價 ×1/×3/×6 後還正 EV 嗎
6. **Matched-null** — 打亂訊號標籤 1000 次，你贏得過隨機嗎
7. **對照基準** — vs 0050 定期定額 / 農場自己的 beta 策略（factor_harvest、GBM 選股）

輸出一份**人話判決書**：這個回測有多少機率是過擬合、扣成本後還剩多少、跟無腦定期定額比誰贏。

## 隱私 & 誠實聲明

- **你的資料不離開瀏覽器。** 所有計算用 [Pyodide](https://pyodide.org)（瀏覽器內的 Python）在你本機跑，CSV 不上傳任何伺服器。
- **這不是投資建議。** 它是統計工具，不推薦任何策略、不叫你買賣。對照組裡的農場策略是「誠實的 beta 產品」當參考基準，不是 alpha、不是推銷。
- **統計通過是必要非充分條件。** 過了七道閘不代表會賺，只代表「沒有明顯的過擬合/挖掘偏誤」。真錢有它自己的風險。

## 技術

純靜態站，零後端。裁判引擎是 Python（numpy/pandas），透過 Pyodide 在瀏覽器內執行；部署在 GitHub Pages。資料處理全程 client-side。
